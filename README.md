# Extensions for Pi Coding Agent

This repository contains extensions for the **Pi Coding Agent**.

These extensions are meant to add extra capabilities and workflows (for example TUI/session helpers, subagent-related tooling, and live workflow widgets) to Pi.

Included in this repo:

- `subagent/` – multi-agent delegation tool with chain/parallel modes
- `sessions-tui/` – session picker command (`/sessions`)
- `kanban-overlay/` – live Kanban widget + `update_tasks` tool + `/kanban`

You can also explore and adapt these extensions for your own experiments with the Pi Coding Agent.
